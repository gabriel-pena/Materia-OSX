# Materia OSX
Materia OSX is aurorae theme that mimics Yosemite window decoration, with Materia colors, based on [Breezemite](https://github.com/andreyorst/Breezemite).
 
## Installation:
Copy theme folders to ~/.local/share/aurorae/themes
